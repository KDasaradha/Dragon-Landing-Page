### **Installation**

1. **Clone the repository**
   ```bash
   git clone https://github.com/kdsr/dragon-landing-page.git
   cd dragon-landing-page
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```

### **Build for Production**
```bash
npm run build
npm run start
```
